## linux配置文件 ##

### 使用方法 ###

    git clone https://github.com/jiazhoulvke/myconf.git ~/.myconf
    cd ~/.myconf/
    ./setup.sh
    
### 文件介绍 ###

#### urxvt配置 ####
* *~/.Xdefault* urxvt配置文件
* *~/.myconf/urxvt/* urxvt的perl扩展

#### bash配置 ####
*.bashrc* bash配置文件

#### tmux配置 ####
* *.tmux.conf* tmux配置文件
* *tmuxstatusline.sh* tmux状态栏脚本

#### screen配置 ####
*.screenrc* screen配置文件
