## 我的linux配置文件 ##

### 一、使用方法 ###

    git clone https://github.com/jiazhoulvke/myconf.git ~/.myconf
    cd ~/.myconf/
    ./setup.sh
    
### 二、文件介绍 ###

#### urxvt配置 ####
* *.Xdefault* urxvt配置文件
* *urxvt/* urxvt的perl扩展目录

#### bash配置 ####
* *.bashrc* bash配置文件

#### tmux配置 ####
* *.tmux.conf* tmux配置文件
* *tmuxstatusline.sh* tmux状态栏脚本

#### screen配置 ####
* *.screenrc* screen配置文件
